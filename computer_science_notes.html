4. Operating Systems

OS manages hardware and software resources of the system.

  • Process Management
  • Memory Management
  • File Systems
  • Scheduling Algorithms: FCFS, SJF, Round Robin

5. Database Management Systems (DBMS)

DBMS allows users to define, create, maintain and control access to databases.

  • SQL Basics: SELECT, INSERT, UPDATE, DELETE
  • Normalization
  • ACID Properties
  • Joins: INNER, LEFT, RIGHT, FULL

6. Computer Networking

Networking connects computers to share resources and information.

  • OSI Model (7 Layers)
  • TCP/IP Model
  • IP Addressing, Subnetting
  • Protocols: HTTP, FTP, DNS, DHCP
